# Two-Stage CMOS Operational Amplifier Design

## Project Overview
This repository contains the formal design specifications and analytical documentation for a two-stage CMOS Operational Amplifier. The design was developed using 0.18µm process parameters, focusing on optimizing open-loop gain, phase margin, and power dissipation.

## Design Parameters & Specifications
The project involved calculating device dimensions (W/L ratios) and biasing currents to meet the following design targets:
- **Process:** 0.18µm CMOS
- **Key Metrics:** Open-loop gain, Unity-Gain Bandwidth (UGBW), and Phase Margin for stability.
- **Compensation:** Analysis of Miller compensation requirements for frequency response.

## Repository Contents
To review the design methodology, please see the following files:

1. **Design_Documentation.pdf:** Contains the circuit topology analysis, small-signal models, and the iterative calculation process used to determine component sizing.
2. **Project_Specifications.pdf:** Outlines the target constraints, bias point summaries, and predicted performance metrics.
